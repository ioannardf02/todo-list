# ToDo-List (Easy Plan)

ToDo list is a simple web application developed in **React + TypeScript + Vite**.

## Description

This app includes the following possibilities:

* Add new Task
* Delete Task
* Completed Task
* Pending Tasks
* Total Tasks
* Search Task
* Dark Mode
* Loading Page
* Not Found Page
* Pagination
* DummyJSON-Fake REST API
